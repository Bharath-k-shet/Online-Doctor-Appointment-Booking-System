# 🩺 Online Doctor Appointment Booking System

A full-stack MERN web application that allows users to book doctor appointments online, manage profiles, and make secure payments using Razorpay.

---

## 🚀 Features

- 👤 User Authentication (Sign up / Login)
- 🏥 Browse Doctors by Speciality
- 📅 Book Appointments with Time Slots
- 💳 Secure Online Payment (Razorpay Integration)
- 📋 View & Manage Appointments
- ❌ Cancel Appointments
- 👤 User Profile Management
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

**Frontend:**
- React.js  
- Redux Toolkit  
- Tailwind CSS  
- Axios  
- React Router DOM  

**Backend:**
- Node.js  
- Express.js  

**Database:**
- MongoDB Atlas (Mongoose)

**Payment Gateway:**
- Razorpay
